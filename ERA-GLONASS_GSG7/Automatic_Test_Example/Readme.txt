Welcome in the ERA-GLONASS testing tool!

To use this tool, recommended versions are:
        - Python 3.10
        - Skydel 22.12 

Python libraries needed are numpy, matplotlib, and cv2.

Download the ERA-GLONASS_GSG8 folder in the Document > Skydel-SDX > API > Python local folder.

Please refer to the ERA-GLONASS application note on Safran website to know more about how to use the tool.
